PROMPT WRITING

เขียน Prompt ยังไงให้ได้ผลลัพธ์ดี

Prompt คือ "คำสั่ง" ที่เราพิมพ์ให้ AI — ยิ่งเขียนชัด AI ยิ่งตอบตรงประเด็น

🌱

1. Context

บอก AI ว่าคุณเป็นใคร กำลังทำอะไร

"ฉันเป็นนักศึกษา กำลังออกแบบเกม..."

🎯

2. สิ่งที่ต้องการ

บอกให้ชัดว่าอยากได้อะไร

"ช่วยออกแบบ Game Mechanic ให้หน่อย"

📋

3. Format

บอก AI ว่าอยากได้คำตอบแบบไหน

"สรุปเป็นตาราง 5 ข้อ"

❌ Prompt ไม่ดี vs ✅ Prompt ดี

// ❌ PROMPT ไม่ดี ช่วยทำเกมให้หน่อย
// ✅ PROMPT ดี ฉันเป็นนักศึกษา Game Design กำลังออกแบบเกม Puzzle Platformer 2D สำหรับเล่นบน PC ช่วยออกแบบ Core Mechanic 3 อย่างที่ทำให้เกมสนุก โดยมี: - กลไกหลักที่ผู้เล่นใช้บ่อย - กลไกรองที่เพิ่มความหลากหลาย - กลไกพิเศษที่ปลดล็อกทีหลัง สรุปเป็นตาราง: ชื่อกลไก / อธิบาย / ตัวอย่างเกมที่ใช้

💡 Pro Tips

• ถ้า AI ตอบไม่ตรง → พิมพ์เพิ่มเติมในแชทเดิมได้ (ไม่ต้องเริ่มใหม่)
• ลองบอก AI ว่า "ช่วยปรับให้สั้นลง" หรือ "ขยายข้อ 2 ให้ละเอียดกว่านี้"
• ใส่ตัวอย่างให้ AI ดู → ยิ่งได้คำตอบที่ตรงใจ

PROMPT PATTERNS

Prompt Templates สำหรับ Game Design

ก็อป Prompt ด้านล่างไปวางใน AI ได้เลย แก้รายละเอียดให้ตรงกับเกมของคุณ

🎯 Pattern 1: Brainstorm ไอเดียเกม

// BRAINSTORM GAME IDEAS ฉันอยากออกแบบเกมแนว [ประเภทเกม เช่น Puzzle / Action / Horror] สำหรับแพลตฟอร์ม [PC / Mobile / Web] กลุ่มเป้าหมายคือ [เด็ก / วัยรุ่น / ทั่วไป] ช่วย brainstorm ไอเดียเกมมา 5 แบบ แต่ละแบบมี: - ชื่อเกม - Concept สั้นๆ 1-2 บรรทัด - จุดเด่นที่แตกต่างจากเกมอื่น - Core Mechanic หลัก

📄 Pattern 2: สร้าง Game Design Document

// GENERATE GDD ช่วยสร้าง Game Design Document สำหรับเกมชื่อ "[ชื่อเกม]" Concept: [อธิบายเกมสั้นๆ] Genre: [แนวเกม] Platform: [แพลตฟอร์ม] Players: [จำนวนผู้เล่น] Target Audience: [กลุ่มเป้าหมาย] ช่วยเขียน GDD ครอบคลุม: 1. Game Overview (concept, unique selling point) 2. Core Gameplay Mechanics 3. Game Flow (เริ่มเกม → เล่น → จบเกม) 4. Level / Difficulty Progression 5. Art Style Direction 6. Technical Requirements

💻 Pattern 3: สร้าง Prototype

// BUILD PROTOTYPE สร้างเกม HTML ที่เล่นบนเว็บได้ ชื่อ "[ชื่อเกม]" กติกา: - [อธิบายวิธีเล่น] - [เงื่อนไขชนะ/แพ้] - [ระบบคะแนน] Gameplay: - ควบคุมด้วย [keyboard / mouse / touch] - สนามขนาด [กว้าง x สูง] - ตัวละครเป็น [รูปแบบ] Design: สไตล์ [Cyberpunk / Retro / Minimal / Pixel Art] ทำเป็นไฟล์ HTML ไฟล์เดียวที่เปิดเล่นได้เลย
CAUTION

ข้อควรระวังในการใช้ AI

AI เป็นเครื่องมือที่เก่งมาก แต่มีข้อจำกัดที่ต้องรู้

💥 AI Hallucination

AI อาจ "แต่งข้อมูล" ที่ดูสมจริงแต่ไม่จริง — ข้อมูลตัวเลข สถิติ ชื่อเกม ต้องตรวจสอบด้วยตัวเองเสมอ

🔍 ต้องทดสอบเสมอ

Code ที่ AI สร้างอาจมี bug — ต้องเทสต์จริง ลองเล่นจริง อย่าเชื่อว่า AI ทำถูก 100%

🔧 AI เป็นเครื่องมือ

AI ช่วยได้เร็ว แต่ไม่ได้แทน Game Designer — ความคิดสร้างสรรค์ การตัดสินใจ ยังเป็นของมนุษย์

⚠️ อย่า Copy-Paste สุ่มสี่สุ่มห้า

อ่านและทำความเข้าใจก่อนใช้ — ถ้าไม่เข้าใจสิ่งที่ AI ตอบ ให้ถาม AI อธิบายเพิ่ม หรือถามอาจารย์

💡 Key Takeaway

ใช้ AI เป็น จุดเริ่มต้น ไม่ใช่คำตอบสุดท้าย
คิดเอง → ให้ AI ช่วย → ตรวจสอบ → ปรับปรุง → ทำให้เป็นของคุณ
