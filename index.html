AI PROTOTYPE

สร้าง Playable Prototype ด้วย Prompt เดียว

จากโจทย์ "Lava Escape" — ใช้ AI generate เป็น HTML Game ที่เล่นได้จริงทันที ด้านล่างคือเกมที่สร้างจาก prompt

// Prompt ที่ใช้สร้างเกม Lava Escape อยากได้เกมชื่อว่า Lava Escape เป็น Game HTML ง่ายๆ โดย กติกา: พื้นที่ในสนามบางจุดจะเปลี่ยนเป็นพื้นลาวาและพื้นที่ปลอดภัย ผู้เล่นจะต้องวิ่งไปยืนในพื้นที่ปลอดภัย เพื่อเอาชีวิตรอดให้ได้นานที่สุด ทุกครั้งที่รอดจากลาวาได้ จะได้รับคะแนน Gameplay สำหรับเล่นบน PC เพื่อทดสอบ POC - จำลองสนาม LED พื้นที่ 10 x 20 ช่อง - ตัวละครผู้เล่นควบคุมด้วย WASD แทนตำแหน่งผู้เล่นด้วยวงกลม มี HP 5 หน่วย - เมื่อเริ่มเกม พื้นในสนามจะกระพริบสีส้มแสดงจุดที่ลาวาจะท่วม 3 วินาที - ผู้เล่นจะต้องวิ่งไปยืนในพื้นที่ปลอดภัยภายในเวลาที่กำหนด - เมื่อครบ 3 วิ จุดที่ลาวาท่วมจะเปลี่ยนเป็นสีแดง จุดที่ปลอดภัยจะเป็นสีเขียว - ถ้าผู้เล่นไม่โดนลาวาจะได้รับ 10 คะแนน ถ้าโดนลาวาจะโดนหัก 10 คะแนน ลองทำเป็นเกมที่เล่นบนเว็ปได้หน่อยนะ เป็น HTML เกม Design สวยๆ หน่อย เอาแนวๆ Cyberpunk ก็ได้

AI PROTOTYPE

Snake Wars — Dual Display POC

จำลองระบบ Fortal Interactive: LED Floor แสดงตำแหน่งผู้เล่น + Projector แสดงเกมงู — 2 ผู้เล่นแข่งกัน!

// Prompt ที่ใช้สร้างเกม Snake Wars สร้างเกม "Snake Wars" เป็น HTML Game สำหรับ 2 ผู้เล่น Concept: จำลองระบบ Fortal Interactive ที่มี 2 จอ - จอซ้าย "LED Floor" (10x20 grid) แสดงตำแหน่งผู้เล่น P1 P2 - จอขวา "Projector" แสดงเกมงูจริง Gameplay: - P1 ควบคุมด้วย WASD, P2 ควบคุมด้วย Arrow Keys - งูกินอาหารแล้วตัวยาวขึ้น - ชนผนัง/ตัวเอง/หางคนอื่น = ตาย - คนสุดท้ายที่รอดชนะ Design แนว Cyberpunk สีเข้ม

WORKSHOP

ลองสร้างเกมของคุณเอง!

โจทย์: ใช้ AI (Gemini / Claude / ChatGPT) สร้าง Prototype Game ง่ายๆ ที่เล่นได้ใน Browser

🐍 เกมงู (Snake)

Classic snake game — ลอง prompt ให้ AI สร้างเกมงูที่เล่นบนเว็บ พร้อม scoring และ speed ที่เพิ่มขึ้น

🎵 เกม Rhythm

กดปุ่มตามจังหวะเพลง — ลองออกแบบ mechanic แล้วให้ AI สร้าง prototype

Tip: เริ่มจากคิด Game Mechanic ก่อน → เขียน Prompt ที่อธิบาย gameplay ชัดเจน → ให้ AI สร้าง HTML → ทดสอบ → แก้ prompt ถ้าต้องปรับ
